# 五子棋 AI - Minimax 版

> **版本**: v1.1 (2026-01-11)  
> **作者**: Elysium-Seeker + AI 辅助增强

---

## 📋 简介

基于 **Minimax 搜索 + Alpha-Beta 剪枝** 的传统五子棋 AI，轻量高效，棋力稳定。

---

## 🚀 快速开始

### 编译
```bash
mingw32-make
```

### 运行
```bash
.\gomoku.exe
```

---

## 🎮 游戏模式

| 模式 | 说明 |
|------|------|
| **1. PvP** | 双人对战 |
| **2. PvE (执黑)** | 玩家先手，AI 后手 |
| **3. PvE (执白)** | AI 先手，玩家后手 |
| **4. EvE** | AI 自我对战 |

**落子**: 输入坐标如 `H8`、`A1`、`O15`

---

## 🧠 AI 算法

| 技术 | 说明 |
|------|------|
| **搜索算法** | Negamax (Minimax 变体) |
| **剪枝** | Alpha-Beta 剪枝 |
| **置换表** | Zobrist Hashing，约 100 万条目 |
| **迭代加深** | 深度 2 → 4 → 6 → ... → 12 |
| **时间控制** | 15 秒超时自动返回 |

### 棋型评估分数

| 棋型 | 分数 |
|------|------|
| 连五 | 10,000,000 |
| 活四 | 1,000,000 |
| 冲四 | 100,000 |
| 活三（含跳三）| 50,000 |
| 眠三 | 5,000 |
| 活二 | 500 |
| 眠二 | 50 |

### 组合威胁加成
- **双冲四** → 相当于活四
- **冲四 + 活三** → 大幅加分
- **双活三** → 相当于冲四
