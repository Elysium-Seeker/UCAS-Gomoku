## 操作说明
1. 游戏开始后，黑棋先手。
2. 输入坐标进行落子，格式为 **字母+数字** (例如 `H8`, `A1`, `O15`)。
   - 字母 A-O 代表列。
   - 数字 1-15 代表行。
   - 输入不区分大小写。
3. 黑棋落子时，程序会自动检查是否为禁手。如果是禁手，黑棋直接判负。
4. 率先连成 5 子的一方获胜（黑棋若同时形成五连和禁手，算黑胜）。
